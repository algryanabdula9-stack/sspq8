# SSP.Q8 — النسخة النظيفة الأولى

هذه نسخة تجريبية ثابتة تعمل على GitHub Pages دون خادم أو قاعدة بيانات حقيقية.

## النطاق الحالي

- إدارة الباقات من لوحة الأدمن دون تعديل الكود.
- دخول العميل عبر OTP تجريبي.
- اختيار الباقة وقراءة تفاصيلها وحدودها.
- إقرار إلزامي قبل الدفع التجريبي.
- إنشاء الطلب ومتابعة حالته.
- مراسلات بين الإدارة والعميل.
- رفع ملفات العميل وملفات التسليم.
- تنزيل وثيقة HTML توثق الطلب والإقرار والتسليم.
- إدارة العملاء والمؤشرات والإعدادات.

## غير موجود حالياً

- بوابة دفع حقيقية.
- رسائل OTP حقيقية.
- قاعدة بيانات سحابية أو مزامنة بين الأجهزة.
- الشركات المشتركة.
- المنتجات الجاهزة.

## التشغيل

1. ارفع جميع الملفات إلى جذر مستودع GitHub Pages.
2. افتح `admin.html` أولاً.
3. أنشئ حساب الإدارة المحلي.
4. عدّل الباقات وحدد السعر والمدة ثم فعّل النشر.
5. افتح `index.html` لتجربة الموقع كعميل.

## تنبيه أمني

بيانات النسخة محفوظة في IndexedDB وLocalStorage داخل المتصفح. حذف بيانات الموقع من المتصفح يمسح الحسابات والطلبات والملفات. هذه النسخة نموذج أولي فقط وليست مناسبة لاستقبال بيانات عملاء حقيقية.
